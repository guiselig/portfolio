Experiência Criativa

1. Desenvolvimento de Jogo Interativo

Objetivo: Desenvolver um jogo em equipe utilizando o ambiente de desenvolvimento Construct.

Descrição: O projeto consistiu na criação de um jogo, promovendo a colaboração em equipe e a divisão de papéis como programador, artista gráfico e game designer para a concepção de um produto coeso e funcional.

Aprendizados: Aplicação prática de metodologias de desenvolvimento de jogos, trabalho em equipe multidisciplinar e utilização da ferramenta Construct para prototipagem e desenvolvimento.

2. Aplicativo Multimídia Educacional

Objetivo: Desenvolver um aplicativo multimídia com foco em "Objetos Educacionais", utilizando o ambiente Processing.

Descrição: O projeto explorou o uso de múltiplas mídias (som, imagem, vídeo e texto) para criar um recurso de aprendizagem eficaz, baseado nos conceitos de Design Instrucional para arquitetar a informação de forma eficiente.

Aprendizados: Programação em Java no IDE Processing , compreensão do Java Runtime Environment (JRE) e aplicação de teorias de Design Educacional na criação de tecnologia para entretenimento e informação.

3. Desenvolvimento de Aplicação Web

Objetivo: Criar um aplicativo web simples , compreendendo os conceitos fundamentais da arquitetura Cliente-Servidor e servindo como um projeto de Extensão Universitária.

Descrição: O projeto visou a difusão do conhecimento gerado na universidade para a comunidade externa , através de um site cujo tema e estratégia foram definidos a partir de entrevistas e validado com o público-alvo.

Aprendizados: Domínio das tecnologias front-end (HTML para estrutura, CSS para estilo e JavaScript para interatividade) e back-end (servidor web Apache e banco de dados MySQL, gerenciados via XAMPP).

Raciocínio Algorítmico

1. Estudo de Caso: Análise Estatística de Turmas

Objetivo: Desenvolver um programa para fornecer uma visão estatística rápida das notas de turmas de tamanho variado.

Descrição: O programa solicita as notas dos estudantes, permitindo inserção contínua , valida os valores para que estejam no intervalo de 0 a 10 e exibe um contador para facilitar o controle. Ao final, apresenta um relatório estatístico completo.

Aprendizados: Lógica de programação para tratamento de dados, incluindo laços de repetição, validação de entrada e cálculo de métricas como média, maiores e menores valores , e percentuais de alunos aprovados, em recuperação e reprovados.

2. Ferramenta de Análise de Sintomas de Burnout

Objetivo: Desenvolver um programa em Python para analisar sintomas de Burnout, calculando o nível de risco e sugerindo ações preventivas.

Descrição: A aplicação processa uma matriz de dados (lista de listas) para calcular um score de risco individual (baixo, moderado ou alto) com base em pesos pré-definidos.

Aprendizados: Manipulação de estruturas de dados complexas em Python (matrizes), implementação de algoritmos para cálculo de scores ponderados e desenvolvimento de lógica condicional para classificação de risco e sugestão de ações.

Fundamentos de Sistemas Ciberfísicos

1. Pesquisa e Seminário sobre Sistemas Operacionais

Objetivo: Realizar uma pesquisa aprofundada sobre os fundamentos de Sistemas Operacionais e produzir um vídeo explicativo sobre um sistema específico sorteado.

Descrição: O trabalho foi dividido em um relatório sobre conceitos (abstração de recursos, gerência, interrupções) e um vídeo para o YouTube sobre um S.O., abordando sua utilidade, sistemas de arquivos, requisitos, pontos fortes e fracos.

Aprendizados: Pesquisa em fontes acadêmicas , redação técnica, produção de conteúdo audiovisual, trabalho em equipe e aprofundamento nos conceitos de arquitetura de sistemas operacionais.

2. Relatório Técnico sobre Operações de Bits

Objetivo: Descrever formalmente as operações de Bitwise e Shift de Bits, com exemplos práticos.

Descrição: O projeto exigiu a elaboração de um documento técnico com estrutura formal (capa, sumário, introdução, etc.) , fundamentado exclusivamente em referências acadêmicas como livros e artigos.

Aprendizados: Pesquisa e escrita acadêmica, e compreensão teórica e prática de operações de baixo nível que são fundamentais para a computação.

3. Proposta de Projeto Final em IoT

Objetivo: Elaborar uma proposta de tema viável para o trabalho final da disciplina, utilizando o microcontrolador ESP32.

Descrição: O processo envolveu a pesquisa de projetos de IoT com ESP32 , estudo de plataformas de nuvem e do protocolo de comunicação MQTT , culminando na elaboração de uma proposta formal de trabalho em equipe.

Aprendizados: Metodologia de pesquisa para projetos de hardware, planejamento e análise de viabilidade técnica, e estudo de tecnologias chave para a Internet das Coisas (IoT), como microcontroladores e protocolos de comunicação.
